# 💬 Java Socket Chat Application

A real-time chat application built using Java Sockets and Swing GUI.  
This project demonstrates client-server communication, multithreading, and basic GUI design.

---

## 🚀 Features
- Real-time messaging using TCP sockets  
- Client-Server architecture  
- Multi-threaded communication (simultaneous read/write)  
- GUI-based chat interface using Java Swing  
- Auto-scroll chat window  

---

## 🛠️ Tech Stack
- Java  
- Socket Programming  
- Multithreading  
- Swing (GUI)
